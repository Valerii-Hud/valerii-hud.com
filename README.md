## 🚀 Run Locally

1. Clone the repository:
   ```bash
   git clone https://github.com/Valerii-Hud/valerii-hud.com.git
   cd valerii-hud.com
   
2. Install dependencies:
   ```bash
    npm install
    ```

3. Start the development server:
    ```bash
    npm run dev
    ```
4. Open in your browser: http://localhost:5173
    ```bash
    http://localhost:5173
    ```
